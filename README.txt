1. DESIGN PHILOSOPHY
------------------------------------------------------------------------
The website embodies a "Luxury Institutional" aesthetic, balancing 
timeless heritage with modern digital precision. The design focuses on 
high-fidelity visual elements, premium textures, and a seamless user 
experience that reflects the craftsmanship of luxury eyewear.

Key Principles:
- Minimalist yet Rich: Using space and typography to convey premium quality.
- Glassmorphism: Subtle blurs and semi-transparent layers for depth.
- Tactile Interactions: Smooth transitions and responsive hover states.


2. COLOR PALETTE
------------------------------------------------------------------------
The site features a dynamic dual-mode color system that maintains 
luxury standards in both light and dark environments.

LIGHT MODE:
- Primary Background (Navy): #FCFAF7 (Off-white/Bone)
- Deep Surface (Navy Deep): #F5F2ED
- UI Surface (Navy Surface): #EBE7E0
- Foreground Text (Cream): #121418 (Deep Obsidian)
- Primary Accent (Gold): #A68A3B

DARK MODE:
- Primary Background (Navy): #0A0E1A (Midnight Navy)
- Deep Surface (Navy Deep): #060810
- UI Surface (Navy Surface): #12182B
- Foreground Text (Cream): #F7F4EF (Off-white)
- Primary Accent (Gold): #C9A84C

COMMON ACCENTS:
- Gold Light: #C9A84C
- Gold Muted: #8B6F2E
- Gold Pale: #F5EDD6
- Teal: #337E77 / #7ECAC3


3. TYPOGRAPHY
------------------------------------------------------------------------
A curated trio of fonts ensures a perfect balance between editorial 
elegance and functional clarity.

- HEADINGS (Serif): Cormorant Garamond
  - Used for large titles, section headers, and quotes.
  - Represents the heritage and "Old World" luxury of the brand.

- BODY TEXT (Sans-serif): Inter
  - Used for descriptions, product details, and UI labels.
  - Provides maximum readability and a clean, technical feel.

- ACCENTS (Serif): Playfair Display
  - Used for specific decorative headings and call-to-actions.

TYPOGRAPHY UTILITIES:
- Letter Spacing (Luxury): 0.25em
- Letter Spacing (Ultra): 0.4em


4. SITE STRUCTURE & PAGES
------------------------------------------------------------------------
The application is built on a modern Next.js 14+ architecture.

MAIN NAVIGATION:
- Home: The central hub featuring the Editorial Hero and Featured Collections.
- Shop: Full product catalog with advanced filtering and categories.
- Brands: Dedicated showcase for curated luxury eyewear houses.
- Magazine: Editorial content and style guides.
- About: The history and vision of Punjab Optical (since 1987).
- Contact: Interactive booking and store locator.

USER FEATURES:
- Virtual Try-On: AR-powered frame testing.
- Wishlist: Personal collection management.
- Cart/Checkout: Seamless shopping experience.
- User Profile: Account management and order history.

ADMINISTRATION:
- Admin Dashboard: Inventory management, product seeding, and analytics.


5. ANIMATIONS & MICRO-INTERACTIONS
------------------------------------------------------------------------
Motion is used to guide attention and add a "living" feel to the interface.

- SMOOTH SCROLLING: Integrated with Lenis for a cinematic feel.
- MARQUEE: Smooth, infinite horizontal scrolling for brand logos (30s cycle).
- TWINKLE: Subtle opacity pulsing for decorative background elements.
- PULSE RING: Interactive scaling effects on focal points.
- LENS FLARE: Dynamic, rotating radial gradients that simulate light.
- FLOAT: Gentle vertical translation (6s cycle) for hero elements.
- SHIMMER: High-end loading states for products and media.
- TRANSITIONS: 500ms ease-on-color and background changes for mode switching.


6. UI COMPONENTS & STYLING
------------------------------------------------------------------------
- GLASS NAVIGATION: Backdrop-blur (25px) with semi-transparent borders.
- PREMIUM CARDS: Gradient backgrounds (135deg) with 10px blur and gold borders.
- CUSTOM SCROLLBAR: Ultra-thin (4px) gold-accented precision scrollbar.
- BACKGROUND GLOW: Soft, dynamic ambient light spheres.
- LOADING SYSTEM: Custom "SpecFrameLoader" for a branded entry experience.
